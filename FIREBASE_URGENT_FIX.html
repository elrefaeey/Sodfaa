🚨 حل عاجل لمشكلة Firebase

⚠️ مشكلة عاجلة: ERR_BLOCKED_BY_CLIENT

هذا يعني أن المتصفح أو إضافة حجب الإعلانات تمنع الاتصال بـ Firebase

🔧 الحلول السريعة (30 ثانية):

  1. إيقاف إضافة حجب الإعلانات: AdBlock, uBlock Origin, etc.
  2. إضافة Firebase للقائمة البيضاء: firestore.googleapis.com
  3. استخدام متصفح آخر: Chrome, Firefox, Edge
  4. وضع التصفح الخفي: Incognito/Private mode

🌐 Chrome

Ctrl + Shift + N (Incognito)

🦊 Firefox

Ctrl + Shift + P (Private)

🌍 Edge

Ctrl + Shift + N (InPrivate)

🔒 Safari

Cmd + Shift + N (Private)
